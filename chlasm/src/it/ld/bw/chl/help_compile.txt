Compile CHL source files into CHL binary file

Syntax
  chlasm -compile -p filename -o filename [-oasm filename]
  chlasm -compile -i files [-h files] [-hi files] -o filename [-oasm filename]

Arguments
  -p    project file
  -i    input source files; order matters
  -h    input c header files
  -hi   input info files
  -o    output CHL file
  -oasm output ASM file (merged)
